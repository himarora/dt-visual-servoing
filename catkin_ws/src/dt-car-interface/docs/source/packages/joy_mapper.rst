joy\_mapper package
===================

.. contents::

The `joy_mapper` package contains the `JoyMapperNode`. This node maps the standard `ROS Joy messages <https://docs.ros.org/kinetic/api/sensor_msgs/html/msg/Joy.html>`_ sent by a (virtual) joystick to actionable items (e.g. start/pause a demo) or robot movement commands.
